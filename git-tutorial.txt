- Some Concept /Commond of git:

    - cloning : To pull all file to your local system
    - Initalize: To Start with Intial Stage:
        - git init : Initalize the folder on git
    - Git Status:
        - It will show you status of File (In which stage it reside currently)
            - Commond : git status
    - Git Add:
        - by help of this commond user can put Untracked file to Staging Area
            - Commond:

                - git add File Name (use when single file)
                - git add -A (Use when Multiple file)


    - Git Commit : 
        - To commit or save file into git server

            - git commit : (press i ->  write  commit message -> :wq )

                                        or

            - git commit -m "Type commit message "


    - Git Checkout :
             - if Accedentally you made a change in your current file and you want to store file which you comment before

                    - Commond:

                        - git Checkout file name(for single file)
                        -  git Checkout -f (for all file)



    - to Close project in git and remove .git file:

        - rm -rf .git(it remove .git folder of current dicrectory)

    - Git .gitignore :
            - It ignore file name which is present Unser .gitignore

    
    - To delete File:
        - git rm filename

    - To Rename File:
        - git mv filename newfile

    - To unTrack some file (if u want that git not track your file)
        - git rm --cached filename

    - To check log : 
        - git log (showing short description of chnage file)
        - git log -p (showing full description of commit file)
        - git log --stat (showing all chnages in short)


    - log filter(google Search :git sem useful option for log filter):

        git log pretty= oneline(print all commit in one line)
        git log pretty = short / full
        git log since=2.days
        git log since= 2.weeks


what if we created new user on git and store ssh to github:

steps:
        1) Create Account
        2) Go to Setting > Shh & GPG Keys
        3) Enter Name and ssh
        to Create SSH key follow Below Steps:
            1) Open your Bash
            2) Paste the text below, substituting in your GitHub email address.
                (ssh-keygen -t rsa -b 4096 -C "Your Email Address")
            3) Adding your SSH key to the ssh-agent:
                ( eval $(ssh-agent -s) )

            4) Add your SSH private key to the ssh-agent. If you created your key with a different name, or if you are adding an existing key that has a different name, replace id_rsa in the command with the name of your private key file.:
                ( ssh-add ~/.ssh/id_rsa )

            5) Copy the SSH key to your clipboard and paste it to github ssh box.
                ( clip < ~/.ssh/id_rsa.pub )

            Now your are Ready to Upload Files.
